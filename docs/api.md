
# Hello API

## `int hello_func(void)`

hello_func 函数

| 参数              | 描述                                |
|:------------------|:------------------------------------|
|`无`               | 无参数                              |
| 返回              | — —                                |
|0                  | 正确执行                            |
|-1                 | 失败                                |

### 例子

``` c
#include <hello.h>

int func()
{
    hello_func();
    
    return 0;
}

```
